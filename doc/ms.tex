\section{Introduction}

The orbit is constrained to the ellipse

\begin{eqnarray}
\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1
\end{eqnarray}

Defining $R=\sqrt{ab}$, and $\epsilon = \sqrt{1 - \frac{b^2}{a^2}}$, this can be
rewritten as
\begin{eqnarray}
x^2 + \frac{y^2}{(1-\epsilon^2)} = a^2
\end{eqnarray}
